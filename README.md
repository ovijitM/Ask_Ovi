# LangChain Project

A basic project structure and starting point for building applications with the LangChain framework in Python.

## Structure
- `src/agents/`: Define custom LangChain agents.
- `src/chains/`: Define complex LCEL chains.
- `src/models/`: Model configurations and initializers.
- `src/prompts/`: Store prompt templates.
- `src/tools/`: Custom tools for agents.
- `src/utils/`: Helper functions.
- `data/`: Local data files or documents.
- `tests/`: Unit tests via pytest.
